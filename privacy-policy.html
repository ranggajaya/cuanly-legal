Privacy Policy for Cuanly

Last updated: August 2026

Cuanly is a personal finance application designed to help users manage wallets, transactions, budgets, and financial reports.

Information We Collect

  • Email address
  • User ID
  • Financial information entered by users such as wallets, transactions, categories, and budgets

How We Use Information

  • Provide app functionality
  • Authenticate users
  • Store and synchronize financial records
  • Improve app security

Data Storage

User data is securely stored using Google Firebase services. Data is encrypted in transit using secure HTTPS connections.

Data Sharing

Cuanly does not sell or share personal user data with third parties for advertising purposes.

Account Deletion

Users can permanently delete their account directly from within the Cuanly application. Deleting an account will permanently remove associated financial records and authentication data.

Security

We take reasonable measures to protect user data from unauthorized access and disclosure.
